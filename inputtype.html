AttributeTYPE in ElementInput

Attribute รายละเอียด
button ใช้เป็นปุ่มใช้งานทั้วไป
checkbox แบบchecklistสัญลักษณ์สี่เหลี่ยม
color รับค่าสีลงไปมีtoolsไว้เลือกสี
date รับค่าเป็นวันที่
email รับค่าในรูปแบบEmailต้องมี@
file รับที่อยู่ของfile**กดที่Browseเพื่ออัปโหลดไฟท์
image เปลี่ยนรูปinputเป็นรูปภาพ***ต้องใส่Attribute src ด้วยนะ
month รับค่าเป็นเดือน***มีtoolsในการเลือกเดือน
number รับค่าเฉพาะตัวเลข โดยรับเลขโดดอย่างเดี่ยวมีtoolsช่าย
password เมื่อพิมพ์ไปตัวอักษรจะถูกคลุมด้วยเครื่องหมายดอกจัน
radio แบบchecklistสัญลักษณ์วงกลม
range แถบเลือนในการinputข้อมูล
reset ล้านข้อมูลทุกElementภายในformเดี่ยวกัน
search รับข้อมูลที่ต้องการค้นหาtext,ตัวเลข
submit ส่งข้อมูลไปปลายทาง
tel รับค่าที่เป็นเบอร์โทรศัพท์
text รับค่าข้อความตัวหนังสือหรือตัวเลขได้
time รับค่าเวลา,มีtoolsช่วยในการกรองข้อมูล
url รับค่าข้อมูลรูปแบบURL
week รับค่าสัปดาห์ มีtoolsในการช่วยinput
